Makeup API Dcumentaion

Makeup Routes

Method URL Body Description
GET /makeup Get a list of all the makeup products, will present 10 results per page
GET /makeup/?page=number Get a list of all makeup products, will show the second page
GET /makeup/?perPage=X X products per 1 page For example 3 products on a page
GET /makeup/search?s=name Makeup search by name For example: lipstick
GET /makeup/category?s=category Makeup search by category For example: Poor makeup
GET /makeup/byPrice?min&max Get a list of all makeup products priced between 10 and 40
GET /makeup/byPrice?min=X &max=Y You can determine the price of the products by entering the desired price in min & max For example: Price between 40 and 100
POST /makeup
  • name required
  • info required
  • category required
  • price required
  • image no required
 Add a new makeup to the data base, You need a token for this query
PUT /makeup/1
  • name required
  • info required
  • category required
  • price required
  • image no required
 Update a makeup in the data base, You need a token for this query
DELETE /makeup/1 Delete a makeup from the data base, You need a token for this query

Users Routes

Method URL Body Description
GET /users
POST /users
  • name required
  • email required
  • password required
 Add a new user to the data base
POST (login) /users/login
  • email required
  • password required
 User login, if the email and password are correct the server will respond with a token
GET (myInfo) /users/myInfo Information -(only for register user with token)
